Lenguaje Multisitio

Es un plugin hecho a la medida para el cambio de idioma del sitio colombia.co

Pasos para configuración: 

1. crear tres paginas para la administración los slug deberian ser. 

/listado-urls-lenguaje/ 
/crear-url/ 
/editar-url/

2. poner sus respectivos shortcode dentro del body de cada pagina. 

Shortcode para listar urls [see_urls]
Shortcode para actualizar url [update_url]
Shortcode para crear url [create_url]
